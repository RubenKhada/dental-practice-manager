# Dental Practice Manager

> Sistema de gestión para una odontóloga independiente que centraliza pacientes, agenda de citas y recordatorios automáticos.

## Problema

Gestionar pacientes y citas de forma manual puede provocar horarios desorganizados, información dispersa y citas olvidadas.

**Dental Practice Manager** busca simplificar la operación diaria de un consultorio odontológico mediante una herramienta clara, organizada y preparada para crecer.

## Usuario objetivo

| Usuario | Uso en la primera versión |
|---|---|
| Odontóloga | Gestiona pacientes, citas y la operación diaria del consultorio. |
| Recepcionista | Rol previsto para una futura versión; no forma parte del alcance inicial. |

## Alcance de la primera versión

| Área | Incluido |
|---|---|
| Pacientes | Registrar, editar y buscar pacientes. |
| Agenda | Crear, consultar, reprogramar y cancelar citas. |
| Horarios | Visualizar horarios disponibles y ocupados. |
| Recordatorios | Enviar recordatorios automáticos de citas. |
| Crecimiento | Diseñar una base que permita añadir una recepcionista en el futuro. |

## Fuera de alcance por ahora

Para mantener una primera versión usable y alcanzable, estas funciones se posponen:

- Portal o aplicación para pacientes.
- Gestión de pagos, facturas e inventario.
- Múltiples odontólogos o consultorios.
- Reportes y analítica avanzada.
- Expediente clínico completo.
- Firma digital, recetas electrónicas o funciones de inteligencia artificial.
